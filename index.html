학급 전체 학생들의 실태 분석 현황과 구글 시트 연동을 관리합니다.

📊 선생님 전용 구글 시트(스프레드시트) 자동 실시간 연동법

학생들이 탐험을 마칠 때마다 구글 스프레드시트에 행으로 이름, 점수, 피드백 등이 지체 없이 자동 저장됩니다. 아래 설정을 한 번만 따라 하시면 됩니다!

1단계. 구글 앱스 스크립트 등록
  1. 구글 스프레드시트를 생성한 뒤 메뉴의 [확장 프로그램] -> [Apps Script]를 누릅니다.
  2. 기존 코드를 다 지우고 하단 복사 버튼을 눌러 코드를 그대로 붙여넣습니다.
  3. 오른쪽 위의 [배포] -> [새 배포]에서 유형을 '웹앱'으로 선택합니다.
  4. 액세스 권한을 가진 사용자를 "모든 사용자(Anyone)"로 설정하고 배포합니다.
  5. 생성된 웹앱 URL을 복사하여 오른쪽 입력창에 넣어 저장해 주세요.
2단계. 구글 웹앱(Apps Script) URL 설정

배포하여 가져온 URL을 기입하고 저장하면 연동 세팅이 완료됩니다. (브라우저 로컬 저장소에 안전하게 유지됩니다)
